## ğŸš€ Why cpbattles?

Traditional competitive programming emphasizes **individual practice**.  
`cpbattles` explores a **battle-oriented approach**, where participants compete
head-to-head on the same problems, encouraging engagement, focus, and learning
through competition.

This repository serves as an **experimental foundation** for evolving
competitive programming battle concepts.
